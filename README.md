# Portfolio

This repository contains a static portfolio site served from `index.html`.

## Deploy on Vercel

1. Open https://vercel.com/new.
2. Import `YogeshReddydev/portfolio`.
3. Keep settings for a static deployment:
   - **Framework Preset:** `Other`
   - **Root Directory:** `/`
   - **Build Command:** leave empty
   - **Output Directory:** leave empty (or `.`)
4. Click **Deploy**.
5. In **Settings → Git**, keep auto-deploy enabled.
6. (Optional) Add a domain in **Settings → Domains**.

`vercel.json` is included in this repo to keep deployment behavior explicit.
